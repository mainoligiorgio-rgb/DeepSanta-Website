# DEEPSANTA — Sito evento

Sito statico dell'evento DeepSanta (Santa Margherita Ligure).

## Avvio locale
Servire la cartella con un web server statico (i font e le immagini sono locali):

    npx serve .

oppure aprire `index.html` con una qualsiasi estensione "Live Server".

## GitHub Pages
Carica il contenuto di questa cartella nella radice del repository, poi attiva
Settings → Pages → Deploy from branch → root. React/Babel sono caricati da CDN
(serve connessione internet).
